How the Power of Nueral Networks can be used to identify key features in a song.

Music has always been my passion, ever since I was a kid I’ve been pounding at thing just to try to make a beat. I would dance whenever I heard any song. Now as an adult that feeling of joy I get from listening to music is still there. Now as a musician I have always been very auditory, while I can read sheet music I’ve always been kind of slow at it. So here my other passion is coming forth. Using my knowledge from previous investigations I thought, this is something that can be done with neural networks. So, I presented the idea to my teammates in the investigation and its was accepted as the project for the fall of 2018.

The final product we want to reach is a system that can listen to a song and automatically transcribes the song in a music sheet. Now systems that do this already exists but they focus on the notes and not the instruments. So, what you are left is a single piano score that captures all the different instruments into one. We want to go beyond this and work on a system that can separate each individual track of the song. Now to work on this we designed a system of three main parts. The first being a neural network that can detect all the instruments on the song and where said instrument is playing. The seconds is going to focus on separating the individual audio of each instrument and third, transcribe these individual tracks into a music sheet.

For the scope of the Fall 2018 semester our work is focused mainly on the first part of the system. A neural network capable of detecting the instruments used in a song. As with any project, especially one using neural networks, one has to start by analyzing how the data is treated. Audio signals have many difficulties when its comes to analyzing, especially when it comes to harmonies. A musical note is created by the resonance of am object on a specific frequency. As an example, for a guitar it’s the strings that vibrate and for our voice it’s the vocal chords that does the magic. Now in addition to the base frequency we also have harmonies. For each instrument the way these harmonies work is unique. This is a key feature we can use to identify which instrument is which. The underlying issue is that when two different instruments play the same note, the waves can be constructive or destructive. This makes it a difficult task to detect what is what. To counter this we need to understand all properties that can make an instrument unique. There are many different ways to approach the instrument recognition. Some utilize fast Fourier transform to work in the frequency domain and other even use raw audio with a CNN. Now since we have experience with Convolutional Neural Networks (CNN). We decided to start working designing the system as such. Though we understand that this system is not really taking into accord all the different features present in an instruments features it gives a quick and easy way to start working.
